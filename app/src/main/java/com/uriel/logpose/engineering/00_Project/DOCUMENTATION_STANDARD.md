# DOCUMENTATION_STANDARD

Version: 1.0

Status: OFFICIAL

------------------------------------------------------------------------------

# PURPOSE

Defines documentation standards.

------------------------------------------------------------------------------

Every module must contain:

README.md

ARCHITECTURE.md

CHANGELOG.md

INTEGRATION.md

------------------------------------------------------------------------------

Every Sprint must update:

CHANGELOG

Relevant README

PROJECT_STATE (if applicable)

------------------------------------------------------------------------------

Documentation must always describe reality.

Documentation must never contradict implementation.

------------------------------------------------------------------------------

If implementation changes, documentation must change.

------------------------------------------------------------------------------

END OF DOCUMENT
