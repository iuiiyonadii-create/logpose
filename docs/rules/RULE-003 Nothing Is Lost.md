# RULE-003 — Nothing Is Lost

## Estado

Vigente

---

# Objetivo

Ninguna idea, decisión o investigación debe perderse.

---

# Regla

Si una idea no forma parte del desarrollo actual:

No se elimina.

Debe documentarse en:

- docs/knowledge/ideas/
- docs/knowledge/future/
- docs/knowledge/rejected/
- docs/knowledge/adr/

según corresponda.

---

# Objetivo

Mantener el conocimiento del proyecto aunque el código cambie.
