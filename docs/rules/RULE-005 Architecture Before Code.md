# RULE-005 — Architecture Before Code

## Estado

Vigente

---

# Objetivo

Toda implementación debe respetar la arquitectura del proyecto.

---

# Regla

Antes de escribir código debe existir un diseño claro.

Si el código contradice la arquitectura,
se revisa la arquitectura antes de continuar.

---

# Principio

La Engineering Bible es la fuente oficial del proyecto.

El código debe seguir la Engineering Bible.

Nunca al revés.
