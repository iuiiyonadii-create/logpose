# Explanation

Toda decisión debe poder explicarse.

Ejemplo

Usuario

Poné algo.

↓

THAMIS

Seleccioné Rock Nacional porque:

- el usuario suele escucharlo al conducir;
- existe un intercom conectado;
- es el horario habitual;
- la confianza fue alta.

------------------------------------------------

Si una decisión no puede explicarse, debe considerarse incorrecta.
