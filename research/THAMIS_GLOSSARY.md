# THAMIS Glossary

Versión 1.0

Este documento define el significado oficial de todos los conceptos utilizados por THAMIS.

------------------------------------------------------------

ACTION

Operación que THAMIS puede ejecutar.

Ejemplo

PLAY_MUSIC

------------------------------------------------------------

INTENT

Objetivo que el usuario intenta comunicar.

Ejemplo

El usuario dice:

"Poné música."

Intent

PLAY_MUSIC

------------------------------------------------------------

ENTITY

Información específica detectada.

Ejemplo

Rock Nacional

Soda Stereo

Volumen

Playlist

------------------------------------------------------------

CONTEXT

Información del entorno.

Ejemplos

Conduciendo

Moto

Lluvia

Auriculares

Intercom

Hora

Ubicación

------------------------------------------------------------

HYPOTHESIS

Posible interpretación.

Puede existir más de una.

------------------------------------------------------------

DECISION

Hipótesis finalmente elegida.

------------------------------------------------------------

KNOWLEDGE

Información conocida por THAMIS.

No pertenece a ningún usuario.

------------------------------------------------------------

PERSONAL KNOWLEDGE

Información aprendida exclusivamente sobre un usuario.

Nunca se comparte.

------------------------------------------------------------

MEMORY

Conjunto de conocimientos personales almacenados.

------------------------------------------------------------

LEARNING

Proceso mediante el cual THAMIS modifica el conocimiento personal.

------------------------------------------------------------

EVIDENCE

Información utilizada para construir una hipótesis.

------------------------------------------------------------

SIMILARITY

Medida de semejanza entre dos expresiones.

------------------------------------------------------------

CONFIDENCE

Nivel de confianza de una hipótesis.

------------------------------------------------------------

POLICY

Regla de funcionamiento.

No representa conocimiento.

Representa comportamiento.

------------------------------------------------------------

RULE

Conocimiento estructurado.

------------------------------------------------------------

CORPUS

Colección de lenguaje utilizada para construir conocimiento.

------------------------------------------------------------

RESEARCH

Proceso mediante el cual se construye el corpus.

------------------------------------------------------------

RUNTIME

Motor que ejecuta THAMIS en el teléfono.

------------------------------------------------------------

COMPILER

Herramienta que transforma Research en Knowledge.

Actualmente no implementado.
