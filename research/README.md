# THAMIS Research

Este directorio contiene la investigación lingüística utilizada para entrenar el conocimiento de THAMIS.

Las expresiones deben provenir del uso real del idioma.

Fuentes permitidas:

- Podcasts
- Streams
- Videos públicos
- Entrevistas
- Conversaciones públicas
- Estudios lingüísticos

No se agregan frases inventadas.

Toda expresión debe clasificarse por:

- País
- Dominio
- Frecuencia
- Fuente
