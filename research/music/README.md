# THAMIS Music Research

## Objetivo

Este directorio contiene la investigación lingüística utilizada para construir el conocimiento musical de THAMIS.

El propósito no es almacenar comandos, sino comprender cómo las personas hablan naturalmente cuando quieren controlar la música.

Todo el conocimiento recopilado aquí será transformado posteriormente en conocimiento estructurado para THAMIS.

---

# Principios

- Nunca inventar expresiones.
- Priorizar lenguaje cotidiano.
- Mantener separados lenguaje, conocimiento y código.
- Registrar regionalismos.
- Documentar ambigüedades.
- Pensar siempre en comprensión, no en coincidencia de texto.

---

# Dominios

Este corpus estudia:

- Reproducir música
- Pausar
- Continuar
- Cambiar canción
- Canción anterior
- Volumen
- Playlist
- Álbum
- Artista
- Género musical
