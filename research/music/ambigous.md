# Expresiones ambiguas

Estas expresiones no tienen una única interpretación.

THAMIS debe utilizar contexto, historial y conocimiento para resolverlas.

------------------------------------------------------------

Expresión

poné algo

Puede significar

- reproducir música
- reproducir podcast
- reproducir radio
- reproducir última playlist

------------------------------------------------------------

Expresión

dale

Puede significar

- continuar
- reproducir
- confirmar una acción

------------------------------------------------------------

Expresión

esa

Puede significar

- repetir canción
- mantener canción actual
- seleccionar canción mencionada anteriormente

------------------------------------------------------------

Expresión

la otra

Puede significar

- siguiente canción
- canción anterior
- otra versión

------------------------------------------------------------

Expresión

subilo

Puede significar

- subir volumen
- subir brillo

------------------------------------------------------------

Expresión

bajalo

Puede significar

- bajar volumen
- bajar brillo

------------------------------------------------------------

Expresión

seguí

Puede significar

- continuar música
- continuar navegación
- continuar llamada

------------------------------------------------------------

Expresión

pará

Puede significar

- pausar música
- cancelar acción
- detener navegación

------------------------------------------------------------

Regla

Nunca ejecutar una acción sensible si existen múltiples interpretaciones con confianza similar.
